### DNS

We recommend using BreakFree DNS or Khoindvn, to ensure installed apps won't get revoked when the certificate in use will expire. 
> NOTE: apps will not expire because they have been installed via this shortcut, but rather because of the certificate they were signed with.

## Features / Roadmap

### Application Sideloading
- [x] `.ipa` file processing
- [x] Direct sideloading without desktop dependencies
- [ ] Support for custom repository links

### Workflow Management
- [ ] Clean, user-friendly Shortcuts prompt interface
- [ ] Post-installation success notifications
- [ ] Error handling for malformed configuration files
- [ ] Batch installation for multiple files simultaneously
